🌍 About Global Azure Day

Global Azure Day is a worldwide community-driven event that brings together developers, IT professionals, and cloud enthusiasts to learn, connect, and explore the latest innovations in Microsoft Azure.

Held across the globe between April 16-18, 2026, this event unites local communities in a shared mission: to share knowledge, showcase real-world solutions, and accelerate the adoption of cloud technologies.

Global Azure Day is powered by passionate local communities, and in Prishtina, it is proudly organized by the Albanian Azure User Group, bringing this global experience to our region.
