# Global Azure Day Prishtina 2026

Welcome to the official website repository for **Global Azure Day Prishtina 2026** - an annual community event dedicated to Microsoft Azure and cloud technologies.

## 🚀 Features

- **Modern Responsive Design**: Beautiful, mobile-friendly website built with HTML5 and CSS3
- **Azure Gradient Theme**: Professional branding with Microsoft Azure colors
- **Smooth Navigation**: Sticky header with easy access to all sections
- **Interactive Sections**:
  - **About**: Event overview and key benefits
  - **Speakers**: Featured industry experts and cloud professionals
  - **Schedule**: Detailed agenda for the event day
  - **Supporters**: Highlighting our partners and sponsors
  - **Location**: Venue information with embedded map

## 🌐 Customization

### Update Event Details
1. Edit `index.html` to add:
   - Specific speakers' names and bios
   - Actual event date and venue
   - Real sponsor/partner information
   - Correct location and venue address

2. Update button colors in `styles.css` if needed

### Add Social Links
Add social media links to the footer in `index.html`

### Modify Colors
The main Azure colors used are:
- Primary Blue: `#0063B1`
- Secondary Blue: `#0078D4`
- Light Blue: `#00A4EF`
- Lighter Blue: `#50E6FF`

## 🚀 Deployment

### Option 1: GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Select `main` branch and `/ (root)` folder
4. Website will be available at `https://yourusername.github.io/global-azure-day-prishtina-2026/`

### Option 2: Azure Static Web Apps
1. Connect GitHub repository to Azure Static Web Apps
2. Click "Create" and follow configuration
3. Website automatically deploys on push

### Option 3: Netlify
1. Connect GitHub repository to Netlify
2. Set build settings (no build required, just serve static files)
3. Deploy and get automatic preview on every push

### Option 4: Traditional Web Host
1. Upload all files to your web server
2. Ensure `index.html` is in the root directory
3. Website will be immediately accessible
